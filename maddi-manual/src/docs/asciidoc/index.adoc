[abstract]
This is the _operator's manual_ for {maddi}: how to run the analyzer on your own project — through
the Gradle plugin, the Maven plugin, or the command line — how to give it the right inputs, how to
read its output, and how to supply analysis hints for libraries. For the *concepts* behind the
annotations it produces (immutability, modification, independence, linking), see the companion
document _The Road to Immutability_.
